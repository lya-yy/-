# 🎯 高二生專注力高效引導網頁系統

這個專案是一個完全基於前端網頁技術（HTML/CSS/JS）開發的專注力引導工具，結合了番茄鐘工作法與分心干擾統計功能，旨在解決高中生自習時容易分心的痛點。

## 🌐 線上體驗網址
👉 [點擊這裡開始專注](https://lya-yy.github.io/-/)

## 📝 專題核心功能
1. **大字體倒數計時**：建立 25 分鐘規律的讀書節奏。
2. **即時分心紀錄**：讀書中途若滑手機、找同學聊天，可一鍵記錄。
3. **數據診斷報告**：專注結束後，自動分析效率並給予客觀的讀書環境建議。

## 🛠️ 使用技術
* HTML5 / CSS3 (採用舒適的深色模式風格)
* JavaScript (異步計時與事件驅動邏輯)
